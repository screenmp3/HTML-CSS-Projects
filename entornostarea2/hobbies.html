Terminal-First Development

My development workflow is heavily terminal-centric. I believe in the power of command-line tools for their speed, efficiency, and scriptability. Most of my daily development tasks happen in the terminal, from code editing to project management.

My primary development languages and tools:

  • Main Languages: Java, Swift, HTML/CSS
  • Primary Editor: Zed with extensive terminal integration
  • Terminal Editor: Neovim for quick edits and remote work
  • Session Management: Tmux for persistent terminal sessions
  • Version Control: Git with custom aliases and workflow
  • Dependencies: Ruby, Python, Node.js (toolchain dependencies)

Advanced Productivity Tools

Beyond basic CLI utilities, I leverage specialized tools that significantly enhance development productivity and system management:

Enhanced File Management

  • lsd - Modern 'ls' replacement with icons, colors, and tree view capabilities
  • tre - Tree command with enhanced output and modern styling
  • ranger - Full-featured terminal file manager with vim keybindings

Package Management & System Tools

  • Homebrew - The cornerstone of my development environment - managing 200+ packages with perfect dependency resolution
  • Raycast - Supercharged Spotlight replacement with extensions for development workflow automation
  • hyperfine - Command-line benchmarking tool for performance testing and optimization

AI-Powered Development

  • Rovo Dev - My primary AI coding assistant, specialized for development workflows and code generation
  • gemini-cli - Terminal interface for Google's Gemini AI, useful for quick queries and code analysis
  • ollama - Local AI model hosting for privacy-focused development assistance
  • Multiple Concurrent AI Agents - Leveraging several AI agents simultaneously in the CLI for complex tasks, allowing for parallel processing of information, diverse problem-solving approaches, and enhanced efficiency in development workflows.

Workflow Integration

These tools work together to create a seamless development experience:

Daily Workflow Example:

  • Project Navigation: Use fzf + zoxide to quickly jump to project directories
  • File Exploration: lsd and tre for beautiful file listings and tree views
  • Code Search: ripgrep + fzf for lightning-fast code searches
  • Performance Testing: hyperfine to benchmark build scripts and optimize compilation times
  • AI Assistance: Rovo Dev for code generation and problem-solving integration
  • Quick Actions: Raycast for project switching, command execution, and workflow automation

My Complete CLI Arsenal

Every CLI tool from my actual Homebrew installation - organized by category:

File Operations & Navigation

  • bat - Enhanced 'cat' with syntax highlighting and Git integration
  • fd - Simple, fast and user-friendly alternative to 'find'
  • fzf - Command-line fuzzy finder for files, history, and processes
  • lsd - Next generation 'ls' command with colors, icons, and tree view
  • ncdu - Disk usage analyzer with ncurses interface
  • ranger - Console file manager with vim-like keybindings and preview features
  • ripgrep - Line-oriented search tool that recursively searches current directory for regex pattern
  • tre-command - Improved tree command with enhanced output and modern styling
  • zoxide - Smart directory jumper that learns your habits

Development & Version Control

  • git - Distributed version control system for tracking changes in source code
  • neovim - Hyperextensible vim-based text editor with modern features and plugin ecosystem
  • tmux - Terminal multiplexer for creating and managing multiple terminal sessions
  • thefuck - Magnificent app that corrects your previous console command

System Monitoring & Performance

  • btop - Resource monitor that shows usage and stats for processor, memory, disks, network and processes
  • fastfetch - Fast system information tool similar to neofetch but written in C
  • htop - Interactive process viewer and system monitor for Unix-like systems
  • hyperfine - Command-line benchmarking tool with statistical analysis
  • speedtest-cli - Command line interface for testing internet bandwidth using speedtest.net

Shell & Terminal Enhancement

  • starship - Minimal, blazing-fast, and infinitely customizable prompt for any shell

Container & Virtualization

  • colima - Container runtimes on macOS with minimal setup
  • docker - Platform for developing, shipping, and running applications in containers

AI & Machine Learning

  • gemini-cli - Command-line interface for Google's Gemini AI models
  • ollama - Get up and running with large language models locally

Build Tools

  • cmake - Cross-platform build system generator
  • ffmpeg - Complete solution to record, convert and stream audio and video
  • gettext - GNU internationalization and localization utilities
  • libtool - Generic library support script for building shared libraries
  • pandoc - Universal document converter between markup formats
  • sqlite - Command-line interface for SQLite databases
  • yt-dlp - Feature-rich command-line audio/video downloader

Other Utilities

  • coderabbit (Cask) - AI-powered code review assistant that integrates with development workflow
  • mactex-no-gui (Cask) - Full TeX Live distribution for macOS without GUI applications
  • sqlcl (Cask) - Oracle SQL Developer Command Line interface for database operations
