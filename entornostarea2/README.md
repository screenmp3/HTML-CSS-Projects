# Development Environments - HTML Assignment

A comprehensive HTML project showcasing terminal-driven development workflow with authentic GitHub dark theme interface.

## Project Overview

This project was created as part of the **Development Environments** course (Unit 02) focusing on installation and use of development environments with HTML markup languages. The assignment demonstrates modern web development practices while showcasing a real CLI-based development workflow with 200+ Homebrew-managed tools.

## Design Philosophy

The project features an **authentic GitHub interface** with:
- **Black top menu bar** (`#000000`) with GitHub Octocat logo and profile dropdown
- **Dark content areas** (`#0e1116`) for sidebar and main content
- **GitHub color palette** with proper contrast ratios
- **Professional profile section** with hover dropdown menu
- **Terminal-focused content** showcasing real CLI development workflow

## Project Structure

```
Project Root
├── index.html              # Main landing page with project overview

```
